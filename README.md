# IPTV Manager для OpenWrt v3.20

Скрипт для настройки IPTV на роутере OpenWrt с веб-админ панелью, встроенным HLS-плеером, EPG телепрограммой и расписанием обновлений.

## Возможности

- **LuCI-плагин** — полная интеграция в веб-интерфейс OpenWrt (Services → IPTV Manager)
- **Управление сервером из LuCI** — запуск, остановка и мониторинг сервера прямо из вкладки «Сервер»
- **Системная информация** — аптайм, оперативная память, свободное место на диске прямо в админке
- **Первоначальная настройка** — пошаговый визард (плейлист, EPG, расписание, автозапуск)
- **Сервер без плейлиста** — запускается сразу, настройка через админку
- Загрузка плейлиста по ссылке (M3U/M3U8), из файла или от провайдера
- **Название плейлиста** — произвольное имя для отображения в статистике
- **Валидация плейлиста** — проверка URL перед загрузкой (доступность + количество каналов)
- **Объединение плейлистов** — загрузка нескольких плейлистов по URL в один файл
- **Авто-обнаружение EPG** — автоматически находит ссылку EPG из заголовка плейлиста (`url-tvg=`)
- **Гибридный EPG** — хранится в RAM (/tmp) в формате gz (~15MB), не занимает флеш-память
- **EPG телепрограмма** — загрузка и раздача XMLTV, авто-распаковка gzip, авто-повтор при ошибке (3 попытки)
- **Проверка каналов** — проверка доступности потоков прямо из админки
- **Избранное** — отмечайте любимые каналы и быстро фильтруйте их
- **Встроенный HLS-плеер** — воспроизведение каналов прямо в браузере с боковым списком каналов, поиском и EPG
- **PiP (Картинка в картинке)** — смотрите канал в маленьком окне и продолжайте работу в браузере
- **3 темы** — светлая, тёмная и OpenWrt Bootstrap Dark (🟣). Авто-определение при встраивании в LuCI
- **Загрузка плейлистов в плеере** — поддержка любых M3U/M3U8 плейлистов
- **Расписание обновлений** — автообновление плейлиста и EPG
- **Безопасность** — пароль на админку (HTTP Basic Auth), API токен, белый список IP, rate limiting
- **Проверка обновлений** — автоматическая проверка новой версии с GitHub
- Веб-админ панель в стиле OpenWrt/Keenetic
- Бэкап и восстановление настроек
- Автозапуск при старте роутера
- Полное удаление одной командой
- Авто-обновление скрипта из GitHub при каждом запуске

## Установка

```sh
wget -q -O - https://raw.githubusercontent.com/whatuneeed/IPTV-Manager/main/IPTV-Manager.sh | sh
```

> Скрипт автоматически обновляется до последней версии с GitHub при каждом запуске.

После установки: `iptv`

## Веб-админ панель

| Страница | URL |
|----------|-----|
| **Админ-панель** | `http://192.168.1.1:8082/cgi-bin/admin.cgi` |
| **Плейлист** | `http://192.168.1.1:8082/playlist.m3u` |
| **EPG** | `http://192.168.1.1:8082/epg.xml` |
| **HLS-плеер** | `http://192.168.1.1:8082/player.html` |

### Вкладки админки

**Каналы** — список каналов с фильтрацией по группам, поиском, проверкой доступности и кнопкой «Смотреть всё»

**Плейлист** — загрузка по ссылке, просмотр исходного M3U, обновление

**Телепрограмма** — настройка EPG, просмотр передач, поддержка XML и XML.gz

**Настройки** — расписание автообновления плейлиста и EPG, бэкап и восстановление, проверка обновлений, пароль, API токен, белый список IP, валидация и объединение плейлистов

## HLS-плеер

Откройте `http://192.168.1.1:8082/player.html` для просмотра каналов прямо в браузере.

### Возможности плеера

- 📺 **Список каналов** — боковая панель с логотипами, группами и поиском
- 📡 **«Сейчас играет»** — текущая передача из EPG отображается в списке каналов
- 🔍 **Поиск** — мгновенная фильтрация по названию канала
- 📂 **Фильтр по группам** — выбор группы из выпадающего списка
- 📋 **Загрузка плейлистов** — вкладка «Плейлист» для загрузки любого M3U/M3U8 по URL
- ⌨️ **Горячие клавиши**:
  - `↑` / `↓` — переключение каналов
  - `Пробел` — пауза / воспроизведение
  - `F` — полный экран
- 🎬 **Полноэкранный режим** — кнопка ⛶ или клавиша F
- 📱 **Адаптивный дизайн** — боковая панель скрывается на мобильных устройствах

### URL-параметры плеера

| Параметр | Описание | Пример |
|----------|----------|--------|
| `url` | Прямая ссылка на поток | `player.html?url=http://...` |
| `idx` | Индекс канала для автостарта | `player.html?url=...&idx=5` |
| `pl` | URL плейлиста для загрузки | `player.html?pl=http://...m3u` |

## Подключение телевизора

В IPTV плеере (TiviMate, IPTV Smarters, VLC, SS IPTV):

- **Плейлист:** `http://192.168.1.1:8082/playlist.m3u`
- **EPG:** `http://192.168.1.1:8082/epg.xml`

## Расписание обновлений

Планировщик работает в фоне и проверяет каждую минуту, пришло ли время обновить:

| Интервал | Когда обновляет |
|----------|-----------------|
| Каждый час | Каждые 60 минут |
| Каждые 6ч | Каждые 6 часов |
| Каждые 12ч | Два раза в сутки |
| Раз в сутки | Раз в 24 часа |

После обновления плейлист и EPG автоматически копируются в `/www/iptv/` — телевизор получает актуальные данные без перезапуска сервера.

## LuCI-плагин (интеграция в OpenWrt)

Плагин `luci-app-iptv-manager` добавляет IPTV Manager в веб-интерфейс OpenWrt как нативный раздел **Services → IPTV Manager**.

### Вкладки плагина

| Вкладка | Описание |
|---------|----------|
| **IPTV** | iframe: полная админка (таблица каналов, проверка, редактирование, плейлист, EPG, настройки) |
| **Плеер** | iframe: HLS-плеер с каналами, полноэкранный режим, PiP |
| **Сервер** | Управление IPTV сервером — запуск, остановка, мониторинг статуса. Работает даже при остановленном сервере (через CGI на порту 80) |

### Управление сервером из LuCI

Вкладка **Сервер** позволяет запускать и останавливать IPTV сервер прямо из LuCI без доступа к терминалу:

- **Статус** — показывает «● Запущен» или «○ Остановлен»
- **Запустить** — запускает uhttpd на порту 8082, генерирует CGI и копирует файлы
- **Остановить** — останавливает uhttpd на порту 8082
- Фоновое CGI (`srv.cgi` на порту 80) гарантирует работу вкладки даже когда основной сервер остановлен

### Установка плагина

**Через скрипт установки** — при первоначальной настройке выберите «Установить LuCI-плагин».

**Вручную:**

```sh
# Скопируйте файлы на роутер:
scp -r luci-app-iptv-manager/htdocs/luci-static/resources/view/iptv-manager/* /www/luci-static/resources/view/iptv-manager/
scp -r luci-app-iptv-manager/root/usr/share/luci/menu.d/ /usr/share/luci/menu.d/
scp -r luci-app-iptv-manager/root/usr/share/rpcd/acl.d/ /usr/share/rpcd/acl.d/
scp -r luci-app-iptv-manager/root/etc/uci-defaults/ /etc/uci-defaults/
/etc/init.d/rpcd restart
```

```

│   └── usr/share/

│       └── rpcd/acl.d/                           # ACL права доступа
```

## Авто-обнаружение EPG

Если в плейлисте в первой строке указана ссылка на EPG (`url-tvg="..."`), скрипт автоматически:
1. Сохранит её как источник EPG
2. Скачает телепрограмму при загрузке плейлиста
3. Отобразит уведомление в админке

## История обновлений

### V1 — Основы
- Базовый скрипт для загрузки плейлистов по URL
- Генерация M3U файла для IPTV устройств
- HTTP-сервер на uhttpd для раздачи плейлиста
- Поддержка HTTPS (--no-check-certificate)
- Автоматическое определение IP роутера
- Минимальная консольная структура

### V2 — Админка и автоматизация
- Веб-админ панель через CGI (admin.cgi) — добавление, удаление, редактирование каналов
- Загрузка плейлиста из файла и настройка IPTV-провайдеров (Xtream Codes)
- Поддержка XMLTV EPG — автообнаружение и загрузка телепрограммы
- Планировщик через crontab — автообновление плейлиста и EPG по расписанию
- Пошаговый визард первоначальной настройки
- Автозапуск при старте роутера через init.d скрипт
- Полное удаление скрипта одной командой
- Подменю SSH для удобства навигации
- Пагинация каналов, логотипы, "Сейчас играет" из EPG

### V3 — Плеер, безопасность и LuCI (текущая v3.20)
- HLS-плеер с боковым списком каналов, поиском и фильтром по группам
- Проверка доступности каналов, избранное, PiP режим
- Подменю Расписание, Бэкап/Восстановление, Безопасность, Обновление
- LuCI-плагин — полная интеграция в веб-интерфейс OpenWrt
- Управление сервером из LuCI (запуск/статус/остановка через srv.cgi на порту 80)
- 3 темы: светлая, тёмная и OpenWrt Bootstrap Dark
- Экспресс-настройка — всё готово за 30 секунд (плейлист по умолчанию + сервер + LuCI)
- Белый список IP, rate limiting, пароль на админку, API токен
- Гибридный EPG — хранение только в RAM (gz формат, ~15MB), без нагрузки на флеш
- Системная статистика: RAM, Диск, Uptime сервера
- Авто-обновление скрипта при каждом запуске
- Сброс к заводским с автозапуском
- Авто-обнаружение EPG из заголовков плейлиста (url-tvg)
- Валидация и объединение нескольких плейлистов
- Полностью переработанное консольное меню
