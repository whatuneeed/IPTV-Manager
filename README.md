# IPTV Manager для OpenWrt v3.2

Скрипт для настройки IPTV на роутере OpenWrt с веб-админ панелью, EPG телепрограммой и расписанием обновлений.

## Возможности

- Загрузка плейлиста по ссылке (M3U/M3U8)
- Загрузка плейлиста из файла
- Настройка IPTV провайдера (логин/пароль)
- **Авто-обнаружение EPG** — автоматически находит ссылку EPG из заголовка плейлиста (`url-tvg=`)
- **EPG телепрограмма** — загрузка и раздача XMLTV, авто-распаковка gzip
- **Проверка каналов** — ping каждого канала прямо из админки
- **Расписание обновлений** — автообновление плейлиста и EPG
- Веб-админ панель в стиле OpenWrt/Keenetic
- Автозапуск при старте роутера
- Полное удаление одной командой

## Быстрый старт

```sh
sh <(wget -O - https://raw.githubusercontent.com/whatuneeed/IPTV-Manager/main/IPTV-Manager.sh)
```

После установки: `iptv`

## Веб-админ панель

| Страница | URL |
|----------|-----|
| **Админ-панель** | `http://192.168.1.1:8082/cgi-bin/admin.cgi` |
| **Плейлист** | `http://192.168.1.1:8082/playlist.m3u` |
| **EPG** | `http://192.168.1.1:8082/epg.xml` |

### Вкладки админки

**Каналы** — список каналов с фильтрацией по группам, поиском и проверкой доступности (пинг)

**Плейлист** — загрузка по ссылке, просмотр исходного M3U, обновление

**Телепрограмма** — настройка EPG, просмотр передач, поддержка XML и XML.gz

**Настройки** — расписание автообновления плейлиста и EPG

## Подключение телевизора

В IPTV плеере (TiviMate, IPTV Smarters, VLC, SS IPTV):

- **Плейлист:** `http://192.168.1.1:8082/playlist.m3u`
- **EPG:** `http://192.168.1.1:8082/epg.xml`

## Расписание обновлений

Планировщик работает в фоне и проверяет каждую минуту, пришло ли время обновить:

| Интервал | Когда обновляет |
|----------|-----------------|
| Каждый час | Каждые 60 минут |
| Каждые 6ч | Каждые 6 часов |
| Каждые 12ч | Два раза в сутки |
| Раз в сутки | Раз в 24 часа |

После обновления плейлист и EPG автоматически копируются в `/www/iptv/` — телевизор получает актуальные данные без перезапуска сервера.

## SSH меню

```
── Плейлист ──────────────────────────
 1) Загрузить по ссылке
 2) Загрузить из файла
 3) Настроить провайдера
 4) Обновить плейлист

── Телепрограмма ─────────────────────
 5) Настроить EPG
 6) Обновить EPG
 7) Удалить EPG

── Сервер ────────────────────────────
 8) Запустить
 9) Остановить

── Настройки ─────────────────────────
10) Расписание обновлений
11) Автозапуск

── Удаление ──────────────────────────
12) Удалить плейлист
13) Удалить IPTV Manager

Enter) Выход
```

## Структура файлов

```
/etc/iptv/
├── iptv.conf           # Конфигурация плейлиста
├── playlist.m3u        # Текущий плейлист
├── provider.conf       # Данные провайдера
├── epg.conf            # Конфигурация EPG
├── epg.xml             # Файл телепрограммы
└── schedule.conf       # Расписание обновлений

/www/iptv/
├── cgi-bin/admin.cgi   # Веб-админка
├── playlist.m3u        # Плейлист для раздачи
└── epg.xml             # EPG для раздачи
```

## Популярные EPG источники

- `https://iptvx.one/EPG_NOARCH` — Россия/СНГ без архива
- `http://epg.it999.ru/epg.xml.gz` — Россия (сжатый)
- `http://epg.pockettv.net/epg.xml.gz` — Россия/СНГ

## Авто-обнаружение EPG

Если в плейлисте в первой строке указана ссылка на EPG (`url-tvg="..."`), скрипт автоматически:
1. Сохранит её как источник EPG
2. Скачает телепрограмму при загрузке плейлиста
3. Отобразит уведомление в админке
